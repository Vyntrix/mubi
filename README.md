## Installation

### Clone the repository

```bash
git clone https://github.com/vyntrix/mubi
```

### Install dependencies

```bash
pnpm i
```

### Setup env variables

```
cp .env.example .env
```

### Push database

```bash
pnpx prisma db push
```

### Start app

```bash
pnpm dev
```
